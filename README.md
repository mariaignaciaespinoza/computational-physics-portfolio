# Computational Physics Portfolio

## Overview

This repository contains a collection of computational physics and numerical methods projects developed during undergraduate studies in Physical Sciences.

The projects focus on scientific computing, mathematical modeling, numerical analysis, differential equations, data analysis, and computational problem solving using Python and LaTeX.

The portfolio documents the implementation and analysis of several classical numerical methods commonly used in physics, engineering, and applied mathematics.

---

## Topics Covered

### Numerical Differentiation

Implementation and analysis of numerical methods for approximating derivatives from discrete data.

### Numerical Integration

Approximation of definite integrals using computational techniques.

### Root-Finding Methods

Numerical algorithms for solving nonlinear equations and locating zeros of functions.

### Differential Equations

Numerical solutions of ordinary differential equations and analysis of dynamical systems.

### Runge–Kutta Methods

Implementation of higher-order methods for solving initial value problems.

### Curve Fitting

Data fitting techniques and model parameter estimation.

### Interpolation

Construction of interpolating functions from discrete datasets.

### Random Processes and Simulations

Computational experiments involving random number generation and stochastic methods.

### Git and Version Control

Introduction to version control practices for scientific computing projects.
