# Source Files

This directory contains the LaTeX source files used to build the Computational Physics Portfolio.

The portfolio documents a collection of numerical methods and scientific computing projects developed in Python and presented as technical reports.

## Directory Structure

### Images

The `img/` directory contains figures, plots, and visualizations generated throughout the projects.

### Introduction

`introduction.tex`

Provides an overview of the portfolio, its objectives, and the computational tools and methodologies employed.

### Projects

Each project is stored as an individual LaTeX file and documents the implementation, analysis, and results of a specific computational physics or numerical methods topic.

Current topics include:

- Numerical Differentiation
- Numerical Integration
- Root-Finding Methods
- Differential Equations
- Runge–Kutta Methods
- Curve Fitting
- Interpolation
- Random Processes and Simulations
- Git and Version Control

### Conclusion

`conclusion.tex`

Summarizes the skills, techniques, and computational methods developed throughout the portfolio.

## Technologies

This portfolio makes use of:

- Python
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook
- Git
- LaTeX

## Purpose

The objective of this portfolio is to demonstrate practical experience in:

- Scientific Computing
- Numerical Analysis
- Mathematical Modeling
- Computational Physics
- Data Analysis
- Technical Documentation

through a collection of computational projects and technical reports.
